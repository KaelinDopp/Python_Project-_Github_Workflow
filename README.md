Yearly Savings Predictor

The program asks the user for their current monthly savings. The program takes the data from the one month and multiplies it by twelve to predict a yearly savings. Lastly, the program displays the final calculation.

The program runs through:
python app.py
# or
python3 app.py

Example interaction:
Enter savings in the current month: 100
You are predicted to save 1200 dollars this year.
